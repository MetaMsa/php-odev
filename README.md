### Kurulum
1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/MetaMsa/php-odev.git
   cd php-odev
2. Veritabanınızı yapılandırın:<br>
    Eğer powershell kullanıyosanız:
    ```bash
    Get-Content 127_0_0_1.sql | mysql -u root -p
    ```
    Eğer cmd kullanıyosanız:
    ```bash
    mysql -u root -p < 127_0_0_1.sql
    ```
